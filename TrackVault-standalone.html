ABOUT TRACKVAULT
TrackVault
A complete single-file DJ toolkit built for real sessions.
No install. No backend. No internet required. Just drop it in a browser.
VERSION 1.0.0  ·  SINGLE-FILE HTML  ·  BROWSER-NATIVE
WHAT'S INSIDE
Music library with drag & drop import
BPM & musical key detection
Waveform visualizer with beat grid
8 hot cue points per track
Camelot wheel harmonic mixing
Dual deck player (A/B)
Manual crossfader
Beat-synced Auto-Mix engine
Pitch & tempo control (±8 / ±16%)
Loop engine — IN/OUT + 1/2/4/8 bar
3-band EQ + kill switches per deck
Frequency spectrum analyzer
Smart library — sort, filter, ratings
Energy scoring & duplicate detection
Drag-to-reorder queue
AI-powered set planner
Session history & play counts
Tap tempo with beat grid rebuild
7-format export (M3U, Rekordbox, Traktor, Serato, CSV, Synapse CSV, Set Sheet)
ID3 tag reader & writer
Full keyboard shortcut set
localStorage persistence — survives refreshes
